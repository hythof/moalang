# The Moa Programming Language
Moa is an open source programming language which helps programming for fun! 


# Getting started

Set up
```
mkdir -p ~/moa/bin
curl https://github.com/moa/bin/moa > ~/moa/bin/moa
export PATH=$PATH:~/moa/bin
```

Create a code
```
# echo 'p "hello world"' > main.moa
```

Run
```
# moa hello.moa | node
hello world
```
